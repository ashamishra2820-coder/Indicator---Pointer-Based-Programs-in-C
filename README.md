# Indicator - Pointer Based Programs in C

## Overview

This project demonstrates the use of pointers in C programming. Two different programs are implemented using pointer concepts. The first program calculates the length of a string without using built-in string functions, while the second program generates cubes of all elements present in a two-dimensional array using a User Defined Function (UDF).

## Objectives

* Understand pointer manipulation in C.
* Practice string processing using pointers.
* Work with two-dimensional arrays.
* Implement User Defined Functions with pointers.

## Program 1: Length Finder

### Description

This program determines the total number of characters present in a string using a pointer. The string is traversed character by character until the null character is reached.

### Sample Output

Input:

```
hello world
```

Output:

```
The length of a string is: 11
```

## Program 2: Cubes Generator

### Description

This program accepts elements of a two-dimensional array and calculates the cube of every element using a User Defined Function and pointers.

### Sample Output

Input:

```
2
3 2
5 4
```

Output:

```
27 8
125 64
```

## Concepts Used

* Pointers
* Arrays
* Two-Dimensional Arrays
* User Defined Functions
* Loops and Conditional Statements
